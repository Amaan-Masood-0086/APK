## ‚öôÔ∏è OPTION 2: Command Line (Advanced)

### Step 1: Android SDK Find Karein

**Method A: Android Studio Se**
1. Android Studio open karein
2. File ‚Üí Settings ‚Üí Appearance & Behavior ‚Üí System Settings ‚Üí Android SDK
3. "Android SDK Location" copy karein

**Method B: Common Locations**
- `C:\Users\YOUR_USERNAME\AppData\Local\Android\Sdk`
- `C:\Android\Sdk`
- `D:\Android\Sdk`

### Step 2: local.properties File Create Karein

Project root mein `local.properties` file banaein:

```properties
sdk.dir=C\:\\Users\\YOUR_USERNAME\\AppData\\Local\\Android\\Sdk
```

**Important:** 
- `YOUR_USERNAME` ko apne Windows username se replace karein
- Backslashes ko double backslash (`\\`) se replace karein

### Step 3: Build APK

```bash
# Clean
.\gradlew.bat clean

# Build Debug APK
.\gradlew.bat assembleDebug

# APK Location
# app\build\outputs\apk\debug\app-debug.apk
```

## ‚ö†Ô∏è TROUBLESHOOTING

### Issue: SDK not found
**Solution:** Android Studio install karein ya SDK manually download karein

### Issue: Build fails
**Solution:** 
- Check `local.properties` file
- Verify SDK path is correct
- Ensure Android SDK Platform 36 installed

### Issue: Gradle sync fails
**Solution:**
- Check internet connection
- File ‚Üí Invalidate Caches / Restart
